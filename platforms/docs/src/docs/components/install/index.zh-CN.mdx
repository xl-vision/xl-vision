## npm

```bash
// 用npm安装
npm install @xl-vision/react styled-components

// 用yarn安装
yarn add @xl-vision/react styled-components
```

## Roboto 字体

在设计 xl-vision 时我们使用了`Roboto`字体。 因此，我们推荐您遵循此说明来载入字体。 例如，通过`Google Web Fonts`引入：

```html
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css?family=Roboto:300,400,500,700&display=swap"
/>
```

## SVG 图标

我们提供的一套 SVG 图标。如果您想使用这个图标，您必须安装`@xl-vision/icons`这个包：

```bash
# 通过 npm
npm install @xl-vision/icons

# 通过 yarn
yarn add @xl-vision/icons
```
