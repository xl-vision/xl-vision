基础的弹出框实现

> 这个组件很强大，使用这个组件包裹的参考组件不会在 dom 上产生任何包裹的元素，因此不会对参考组件产生任何影响。这个组件的弹出框内容只在真正触发时才会挂载，因此性能有一定的保证。

## 属性

> 使用此组件，需要子组件暴露`ref`实例和`onClick`、`onMouseEnter`、`onMouseLeave`、`onFocus`、`onBlur`、`onContextMenu`事件。

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| children | 子元素 | React.ReactElement | — | true |
| popup | 弹出框 | React.ReactElement | - | true |
| placement | 指定弹出框的方位 | `auto` \| `auto-start` \| `auto-end` \| `top` \| `top-start` \| `top-end `\| `right` \| `right-start` \| `right-end` \| `bottom` \| `bottom-start` \| `bottom-end` \| `left` \| `left-start` \| `left-end` | `auto` | false |
| popupContainer | 指定弹出框的父容器 | string \| HTMLElement \| (() => string \| HTMLElement) | () => body | false |
| open | 是否显示弹出框 | boolean | - | false |
| defaultOpen | 默认弹出框是否展示，当`open`存在时，选项失效 | boolean | false | false |
| onOpenChange | 弹出框状态改变时的回调 | (open: boolean) => void | - | false |
| trigger | 触发方式 | `hover` \| `focus` \| `click` \| `contextMenu` \| `false` \| array | `hover` | false |
| disablePopupEnter | 禁止鼠标进入弹出框 | boolean | - | false |
| transitionClassName | 动画 class，参看[CssTransition](/components/CssTransition) | string | - | false |
| className | 弹出框 class | string | 0 | false |
| onAfterClosed | popup 关闭完成时的回调 | () => void | - | false |
| mountOnShow | 在弹出框展示时才挂载 dom | boolean | true | false |
| unmountOnHide | 在弹出框隐藏时才卸载 dom | boolean | - | false |
| hoverOptions | 控制 hover 行为的选项, 参看[usePopper](/hooks/usePopper) | HoverOptions | - | false |
| clickOptions | 控制 click 行为的选项, see [usePopper](/hooks/usePopper) | ClickOptions | - | false |
| focusOptions | 控制 focus 行为的选项, see [usePopper](/hooks/usePopper) | FocusOptions | - | false |
| contextMenuOptions | 控制 宠 textMenu 行为的选项, see [usePopper](/hooks/usePopper) | ContextMenuOptions | - | false |
| shiftOptions | 控制 popper 遇到遮挡时的偏移行为, see [usePopper](/hooks/usePopper) | ShiftOptions \| false | - | false |
| autoPlacementOptions | 控制 popper 遇到遮挡时切换定位方式的行为, see [usePopper](/hooks/usePopper) | AutoPlacementOptions \| false | - | false |
| arrowOptions | 控制箭头的行为, see [usePopper](/hooks/usePopper) | ArrowOptions | - | false |
| autoUpdateOptions | 控制如何自动定位, see [usePopper](/hooks/usePopper) | AutoUpdateOptions | - | false |
| arrow | 箭头 | React.ReactNode | - | false |
| offset | 偏移量, see [usePopper](/hooks/usePopper) | OffsetOptions | 0 | false |
