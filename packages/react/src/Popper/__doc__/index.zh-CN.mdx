# 基础弹出框

基础的弹出框实现

> 这个组件很强大，使用这个组件包裹的参考组件不会在 dom 上产生任何包裹的元素，因此不会对参考组件产生任何影响。这个组件的弹出框内容只在真正触发时才会挂载，因此性能有一定的保证。

## Popper API

> 使用此组件，需要子组件暴露`ref`实例和`onClick`、`onMouseEnter`、`onMouseLeave`、`onFocus`、`onBlur`、`onContextMenu`事件。

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| children | 子元素 | React.ReactElement | — | true |
| popup | 弹出框 | React.ReactElement | - | true |
| placement | 指定弹出框的方位 | `auto` \| `auto-start` \| `auto-end` \| `top` \| `top-start` \| `top-end `\| `right` \| `right-start` \| `right-end` \| `bottom` \| `bottom-start` \| `bottom-end` \| `left` \| `left-start` \| `left-end` | `auto` | false |
| popupContainer | 指定弹出框的父容器 | string \| HTMLElement \| (() => string \| HTMLElement) | () => body | false |
| visible | 是否显示弹出框 | boolean | - | false |
| defaultVisible | 默认弹出框是否展示，当`visible`存在时，选项失效 | boolean | false | false |
| onVisibleChange | 弹出框状态改变时的回调 | (visible: boolean) => void | - | false |
| hideDelay | 延时关闭 | number | - | false |
| showDelay | 延时显示 | number | - | false |
| trigger | 触发方式 | `hover` \| `focus` \| `click` \| `contextMenu` \| `custom` \| array | `hover` | false |
| disablePopupEnter | 禁止鼠标进入弹出框 | boolean | - | false |
| transitionClasses | 动画 class，参看[CssTransition](/components/CssTransition) | boolean | - | false |
| arrow | 箭头 | React.ReactElement | - | false |
| offset | 弹出框偏移 | number | 0 | false |
| className | 弹出框 class | string | 0 | false |
| flip | [参考 popperjs](https://popper.js.org/docs/v2/modifiers/flip/) | boolean \| object | true | false |
| preventOverflow | [参考 popperjs](https://popper.js.org/docs/v2/modifiers/prevent-overflow/) | boolean \| object | true | false |
| transformOrigin | 自动计算 transformOrigin 位置 | boolean | true | false |
| onAfterClosed | popup 关闭完成时的回调 | () => void | - | false |
| mountOnShow | 在弹出框展示时才挂载 dom | boolean | true | false |
| unmountOnHide | 在弹出框隐藏时才卸载 dom | boolean | - | false |
