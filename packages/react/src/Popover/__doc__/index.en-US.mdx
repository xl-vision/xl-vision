## Popover API

> For more parameters, please refer to [Popper](/components/Popper) properties

| Property | Description | Type | Default | Require |
| --- | --- | --- | --- | --- |
| content | Content | React.ReactNode | - | true |
| title | Title | React.ReactNode | - | false |
| transitionClassName | Animation class, will override default animation, see [CssTransition](/components/CssTransition) | string | `xl-popover` | false |
| popupContainer | Specify the parent container of the pop-up box | string \| HTMLElement \| (() => string \| HTMLElement) | () => document.body | false |
| className | Pop-up box class name | string | - | false |
| trigger | Trigger method | `hover` \| `focus` \| `click` \| `contextMenu` \| `custom` \| array | `click` | false |
| showArrow | Show arrow | boolean | - | false |
