## 属性

> 更多参数请参考[Popper](/components/Popper)属性

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| content | 内容 | React.ReactNode | - | true |
| title | 标题 | React.ReactNode | - | false |
| popupContainer | 指定弹出框的父容器 | string \| HTMLElement \| (() => string \| HTMLElement) | () => document.body | false |
| className | 弹出框类名 | string | - | false |
| hideArrow | 是否隐藏箭头 | boolean | - | false |
