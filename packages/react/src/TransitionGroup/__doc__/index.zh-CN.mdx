## TransitionGroup API

> 其他参数见**[CssTransition](/components/CssTransition)**

> **NOTE**

> TransitionGroup 并不支持`appear`和`disappear`钩子和相应的`class`。

| 参数              | 说明                      | 类型                 | 默认值 | 是否必填 |
| ----------------- | ------------------------- | -------------------- | ------ | -------- |
| transitionClasses | 指定动画过程中的`class`名 | string \| object     | -      | false    |
| children          | 数组或单个元素            | React.ReactElement[] | -      | true     |
