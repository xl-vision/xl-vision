## TransitionGroup API

> See [CssTransition](/components/CssTransition) for other properties.

> **NOTE**

> TransitionGroup does not support the `appear` and `disappear` hooks and the corresponding `className`.

| Property | Description | Type | Default | Require |
| --- | --- | --- | --- | --- |
| transitionClasses | Specify the class name during the animation | string \| object | - | false |
| children | Child nodes | React.ReactElement[] | - | true |
