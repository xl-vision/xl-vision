## Button API

> For other parameters, please refer to [BaseButton](/components/BaseButton)

| Property | Description | Type | Optional Value | Default | Require |
| --- | --- | --- | --- | --- | --- |
| href | Click the jump address, the button behavior is the same as the a tag | string | - | - | false |
| disabled | Disable State | boolean | - | false | false |
| color | Set button theme color | string | default &#x007c; primary &#x007c; success &#x007c; warning &#x007c; error &#x007c; secondary | default | false |
| variant | Set button type | string | contained &#x007c; outlined &#x007c; text | contained | false |
| long | Long button | boolean | - | false | false |
| prefixIcon | Prefix icon | ReactElement | - | - | false |
| suffixIcon | Suffix icon | ReactElement | - | - | false |
| loading | Set the button as loading state | boolean | - | - | false |
| round | Round button | boolean | - | - | false |
| disableElevation | Shadow effect is forbidden, only valid for `variant=contained` | boolean | - | - | false |
| children | Child node, can be empty when `prefixIcon` or `suffixIcon` is specified | ReactNode | - | - | false |
| size | Button Size | middle &#x007c; large &#x007c; small | - | middle | false |
