用于跳转到页面指定位置。

## 示例

:::

:::demo ./AffixCn

## 属性

### Anchor

> 启用`affix`时，效果同[Affix](/Affix)

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| affix | 是否固定 | boolean | true | false |
| affixTarget | 固定时参考的元素，不传时和`scrollTarget`相同 | Window \| HTMLElement \| (() => Window \| HTMLElement) | () => window | false |
| scrollTarget | 滚动时的容器，不传时和`scrollTarget`相同 | Window \| HTMLElement \| (() => Window \| HTMLElement) | () => window | false |
| offsetTop | 参考[Affix](/components/Affix) | number | - | false |
| offsetBottom | 参考[Affix](/components/Affix) | number | - | false |
| bounds | 锚点区域边界 | number | 5 | false |
| targetOffset | 锚点滚动偏移量 | number | 0 | false |
| onChange | 监听锚点链接改变 | (currentActiveLink: string) => void | - | false |

### AnchorLink

| 参数  | 说明     | 类型   | 默认值 | 是否必填 |
| ----- | -------- | ------ | ------ | -------- |
| title | 标题     | string | -      | true     |
| href  | 锚点链接 | string | -      | true     |

## 方法

### Anchor

| 方法     | 说明               | 类型                         | 默认值 |
| -------- | ------------------ | ---------------------------- | ------ |
| scrollTo | 滚动到指定的链接处 | (activeLink: string) => void | -      |
