# CollapseTransition

可以方便地对于不定高度或者宽度的内容进行折叠或者展开的动画组件。

:::demo ./Basic.tsx

基础用法

通过`CollaseTrannsition`包裹对应的元素来触发折叠和展开的动画效果。通过`transitionClassName`自定义动画方式。如果不指定，将使用默认动画`width/height 1s ease`

:::

## CollapseTransition API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| in | 触发进出场动画,`true`表示进场，`false`表示出场 | boolean | - | — | true |
| transitionClasses | 指定动画过程中的`class`名,参考<Link to="CSSTransition">`CSSTransition`</Link> | string | - | - | false |
| children | 子元素 | React.ReactElement | - | — | true |
| horizontal | 是否横向 | boolean | - | — | - |
| transitionOnFirst | 是否初始化完成时就触发动画 | boolean | - | false | false |
| mountOnEnter | 当第一次进场动画时才挂载子节点，见<Link to="Transition">**Transition**</Link> | boolean | - | false | false |
| unmountOnLeave | 当出场动画后就卸载子节点，见<Link to="Transition">**Transition**</Link> | boolean | - | false | false |
