## 属性

> Transition 继承所有[useCssTransition](/hooks/useCssTransition)属性

| 参数          | 说明                   | 类型    | 默认值 | 是否必填 |
| ------------- | ---------------------- | ------- | ------ | -------- |
| mountOnEnter  | 第一次执行进入过渡时才挂载子元素，后续会保持挂载状态，除非使用`unmountOnExit` | boolean | -      | false    |
| unmountOnExit | 离开过渡时会卸载子元素 | boolean | -      | false    |
