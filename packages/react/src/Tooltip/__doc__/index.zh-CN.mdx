## Tooltip API

> 更多参数请参考[Popper](/components/Popper)属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| content | tooltip 内容 | React.ReactNode | - | - | true |
| maxWidth | tooltip 最大宽度 | number | - | - | false |
| transitionClassName | 动画 class，会覆盖默认动画，参看[CssTransition](/components/CssTransition) | string | - | 'xl-tooltip' | false |
| bgColor | 自定义背景色 | string | - | - | false |
| trigger | 触发方式 | 'touch'&#x007c;'hover'&#x007c;'focus'&#x007c;'click'&#x007c;'contextMenu'&#x007c;'custom'&#x007c;array | - | ['hover','touch'] | false |
| showArrow | 是否显示箭头 | boolean | - | - | false |
| touchShowDelay | 触屏延迟展示时间 | number | - | 1500 | false |
| touchHideDelay | 触屏延迟隐藏时间 | number | - | 0 | false |
