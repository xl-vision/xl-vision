## Tooltip API

> 更多参数请参考[**Popper**](/Popper)属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| content | tooltip 内容 | React.ReactNode | - | - | true |
| maxWidth | tooltip 最大宽度 | number | - | - | false |
| transitionClassName | 动画 class，参看[**CSSTransition**](/CSSTransition) | string | - | `xl-tooltip-slide` | false |
| getPopupContainer | 指定弹出框的父容器 | string &#x007c; HTMLElement &#x007c; (() => string &#x007c; HTMLElement) | - | 默认为`document.body` | false |
| className | 弹出框类名 | string | - | - | false |
| bgColor | 自定义背景色 | string | - | - | false |
