## Tooltip API

> For more parameters, please refer to [Popper](/components/Popper) properties.

| Property | Description | Type | Default | Require |
| --- | --- | --- | --- | --- |
| content | Tooltip content | React.ReactNode | - | true |
| maxWidth | Tooltip max width | number | - | false |
| transitionClassName | Animation class name, will override default animation, see [CssTransition](/components/CssTransition) | string | `xl-tooltip` | false |
| bgColor | Custom background color | string | - | false |
| trigger | Trigger method | `touch` \| `hover` \| `focus` \| `click` \| `contextMenu` \| `custom` \| array | [`hover`,`touch`] | false |
| showArrow | Show arrow | boolean | - | false |
| touchShowDelay | Delayed display time on touch screen | number | 1500 | false |
| touchHideDelay | Delayed hidden time on touch screen | number | 0 | false |
