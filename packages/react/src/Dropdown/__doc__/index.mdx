# Dropdown API

> 更多参数请参考[**Popper**](/Popper)属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| children | 子元素 | React.ReactElement | - | - | true |
| menus | 下拉菜单 | React.ReactNode | - | - | true |
| transitionClassName | 动画 class，参看[**CSSTransition**](/CSSTransition) | string | - | `xl-dropdown` | false |

# DropdownSubmenu API

> 更多参数请参考[**Popper**](/Popper)属性

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| children | 子元素 | React.ReactElement | - | - | true |
| title | 标题 | React.ReactNode | - | - | true |
| transitionClassName | 动画 class，参看[**CSSTransition**](/CSSTransition) | string | - | `xl-dropdown-submenu` | false |
| disabled | 禁用 | boolean | - | - | false |

# DropdownItem API

| 参数     | 说明     | 类型               | 可选值 | 默认值 | 是否必填 |
| -------- | -------- | ------------------ | ------ | ------ | -------- |
| children | 子元素   | React.ReactElement | -      | -      | true     |
| disabled | 禁用     | boolean            | -      | -      | false    |
| onClick  | 点击事件 | function           | -      | -      | false    |
