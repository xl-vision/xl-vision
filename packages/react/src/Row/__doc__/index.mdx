# 栅格系统

参考`bootstrap`的栅格系统

::: demo ./Basic.tsx

基础用法

使用`span`,`gutter`实现基本的布局（`span`设置为 0 可以实现内容块的隐藏)。

:::

export default () => {

              sm: 2,
              md: 3,
              lg: 4,
              xl: 5,
              xxl: 6,
            }}
          >
            <div className='col-box'>col2</div>
          </Row.Col>
          <Row.Col span={6}>
            <div className='col-box'>col3</div>
          </Row.Col>

        <Row gutter={10}>
          <Row.Col span={6}>

## Row API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| gutter | 内容块间隙(支持响应式) | number &#x007c; object | >=0 | 0 | false |
| type | 布局类型 | flex | - | - | false |
| justify | 栅格左右对齐方式(在`type=flex`下有效) | string | start &#x007c; end &#x007c; center &#x007c; space-around &#x007c; space-between | — | false |
| align | 栅格上下对齐方式(在`type=flex`下有效) | string | top &#x007c; middle &#x007c; bottom | — | false |
| children | `Col`数组 | Col &#x007c; Col[] | - | - | true |

## Col API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| clsPrefix | 自定义类名前缀 | string | - | - | false |
| span | 栅格占据的列数(支持响应式)(**0 表示隐藏**) | number &#x007c; object | 0-24 | — | true |
| offset | 栅格左侧的间隔列数(支持响应式) | number &#x007c; object | 1-24 | — | false |
| push | 栅格向右移动的列数(支持响应式) | number &#x007c; object | 1-24 | — | false |
| pull | 栅格向左移动的列数(支持响应式) | number &#x007c; object | 1-24 | — | false |
| order | 栅格顺序，flex 布局模式下有效(支持响应式) | number &#x007c; object | 1-24 | — | false |
| children | 子节点 | ReactNode | - | - | false |
