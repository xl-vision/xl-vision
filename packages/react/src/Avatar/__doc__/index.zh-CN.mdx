## 属性

### Avatar

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| children | 内容，需要是字符串 | string | - | false |
| icon | 图标 | React.ReactNode | - | false |
| shape | 形状 | `circle` \| `square` \| `round` | `circle` | false |
| size | 尺寸，可以自定义大小 | `small` \| `middle` \| `large` \| number | `middle` | false |
| src | 设置图片地址或图片组件实例 | string \| React.ReactNode | - | false |
| srcSet | 图片的 srcSet | string | - | false |
| onError | 图片加载失败回调，默认图片加载失败会不显示图片，返回 false 时将始终展示图片 | function | - | false |
| alt | 图片的 alt | string | - | false |
| gap | 内容和边框的间隙 | number | 4 | false |

### AvatarGroup

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| children | 头像数组 | ReactElement[] | - | true |
| popupPlacement | 弹出框展示位置,none 表示不展示 | `none` \| `top` \| `bottom` | `top` | false |
| shape | 形状，统一设置头像形状 | `circle` \| `square` \| `round` | `circle` | false |
| size | 尺寸，统一设置头像大小 | `small` \| `middle` \| `large` \| number | `middle` | false |
| maxCount | 最多展示头像数，默认全部展示 | number | - | false |
| maxStyle | 多余头像样式 | React.CssProperties | - | false |
