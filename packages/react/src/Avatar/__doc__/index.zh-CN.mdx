## Avatar API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| children | 内容，需要是字符串 | string | - | - | false |
| icon | 图标 | React.ReactNode | - | - | false |
| shape | 形状 | string | circle &#x007c; square &#x007c; round | circle | false |
| size | 尺寸，可以自定义大小 | string &#x007c; number | small &#x007c; middle &#x007c; large | default | false |
| src | 设置图片地址或图片组件实例 | string &#x007c; React.ReactNode |  | - | false |
| srcSet | 图片的 srcSet | string |  | - | false |
| onError | 图片加载失败回调，默认图片加载失败会不显示图片，返回 false 时将始终展示图片 | function |  | 4 | false |
| alt | 图片的 alt | string |  | - | false |
| gap | 内容和边框的间隙 | number |  | 4 | false |

## AvatarGroup API

| 参数 | 说明 | 类型 | 可选值 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- | --- |
| children | 头像数组 | ReactElement[] | - | - | true |
| popupPlacement | 弹出框展示位置,none 表示不展示 | string | none &#x007c; top &#x007c; bottom | top | false |
| shape | 形状，统一设置头像形状 | string | circle &#x007c; square &#x007c; round | circle | false |
| size | 尺寸，统一设置头像大小 | string &#x007c; number | small &#x007c; middle &#x007c; large | default | false |
| maxCount | 最多展示头像数，默认全部展示 | number |  | - | false |
| maxStyle | 多余头像样式 | CssProperties |  | - | false |
