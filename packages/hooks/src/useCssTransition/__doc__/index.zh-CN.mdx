## usePopper API

| 参数 | 说明 | 类型 | 默认值 | 是否必填 |
| --- | --- | --- | --- | --- |
| placement | popper 的位置 | `top` \| `top-start` \| `top-end `\| `right` \| `right-start` \| `right-end` \| `bottom` \| `bottom-start` \| `bottom-end` \| `left` \| `left-start` \| `left-end` | - | true |
| mode | 定位模式 | `fixed` \| `absolute` | `fixed` | false |
| middlewares | 中间件 | Array\<Middleware\> | - | false |
